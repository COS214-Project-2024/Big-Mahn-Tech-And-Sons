\doxysubsection{Detailed Description}
Interface for creating building objects of various types. 

\mbox{\hyperlink{classBuildingCreator}{Building\+Creator}} defines an interface for generating buildings. Concrete creator classes (e.\+g., \mbox{\hyperlink{classResidentialBuildingCreator}{Residential\+Building\+Creator}}) implement this interface to create specific types of buildings based on input.
