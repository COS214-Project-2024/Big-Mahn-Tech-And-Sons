\doxysubsection{Detailed Description}
A concrete command that allows a citizen to travel from their current position to a random entertainment node. 

This class inherits from \mbox{\hyperlink{classGoToCommand}{Go\+To\+Command}} and encapsulates the logic required for a citizen to navigate to an entertainment location within the city.
