\doxysubsection{Detailed Description}
Represents an event command that boosts economic performance. 

This class inherits from \mbox{\hyperlink{classEventsCommand}{Events\+Command}} and is responsible for implementing the logic that increases the economic performance of a city by benefiting its citizens and commercial buildings during a specific event.
