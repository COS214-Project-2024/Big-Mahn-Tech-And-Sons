\doxysubsection{Detailed Description}
A concrete command class that allows a citizen to travel from their current position to their work destination. 

This class is part of the Command design pattern and enables the simulation of a citizen\textquotesingle{}s journey to work within the road network. Upon successful arrival at the work destination, the citizen may earn income.
