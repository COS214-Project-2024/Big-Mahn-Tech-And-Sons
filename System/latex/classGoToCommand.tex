\doxysubsection{Detailed Description}
Command class for handling a citizen\textquotesingle{}s movement to a specified destination using a transport strategy. 

\begin{DoxyAuthor}{Author}
Lebo\+Gang
\end{DoxyAuthor}
The \mbox{\hyperlink{classGoToCommand}{Go\+To\+Command}} class encapsulates the behavior of a citizen traveling from their current position to a designated destination node. It uses the transport strategy defined by the \mbox{\hyperlink{classCitizen}{Citizen}} object and finds the appropriate path using the Road\+Network.

{\footnotesize\ttfamily virtual bool Go\+To\+Command\+::execute (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [pure virtual]}}



Executes the command to move the citizen to the specified destination. 

The function retrieves the citizen\textquotesingle{}s current position and checks if the destination has already been reached. If not, it uses the transport strategy to move the citizen along a path found in the road network. If no path is available, a message is displayed.

\begin{DoxyNote}{Note}
destination\+Node is set in the constructors of the relevant command. 
\end{DoxyNote}
\begin{DoxyAttention}{Attention}
Citizens must have a catalog of addresses, such as Work, Home, and random Entertainment then find\+Path will do the rest. 
\end{DoxyAttention}
