\doxysubsection{Detailed Description}
Represents a citizen in the life simulation game. 

A \mbox{\hyperlink{classCitizen}{Citizen}} can travel between buildings using various modes of transport. The \mbox{\hyperlink{classCitizen}{Citizen}}\textquotesingle{}s behavior changes based on their state (Child, Adult, Pensioner).
