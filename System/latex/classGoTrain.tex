\doxysubsection{Detailed Description}
A concrete command class that allows a citizen to travel from their current position to a train station. 

This class is part of the Command design pattern and simulates the citizen\textquotesingle{}s journey to a train station within the road network.

Executes the command to move the citizen to the specified destination. 

The function retrieves the citizen\textquotesingle{}s current position and checks if the destination has already been reached. If not, it uses the transport strategy to move the citizen along a path found in the road network. If no path is available, a message is displayed.
