\doxysubsection{Detailed Description}
Command class for handling recession events within the city. 

This class is responsible for managing the economic impact of recession events, including financial adjustments, citizen satisfaction management, and the status of commercial buildings. It coordinates actions across various departments to mitigate the negative effects of a recession.
